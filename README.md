# Blackjack Game

## 📌 Overview
A Python command-line implementation of the classic Blackjack card game. Players compete against the computer by drawing cards and trying to reach a total of 21 without exceeding it.

## ✨ Features
- Play against the computer
- Random card dealing
- Automatic score calculation
- Blackjack and bust detection
- Play multiple rounds

## 🛠️ Technologies Used
- Python 3

## 🚀 How to Run

1. Clone the repository:
```bash
git clone https://github.com/satyam-katara/blackjack-game.git
```

2. Navigate to the project:
```bash
cd blackjack-game
```

3. Run the project:
```bash
python main.py
```
